Alle bestanden mogen alleen als root worden uitgevoerd (of sudo)enkel deploy_bridge is beschikbaar voor 1 enkele andere user (hosting-ftp?) dmv sudo deploy_bridgeapache.sh 			Maakt Vhost aan en herlaad apache met nieuwe vhost indien in ordedeploy_bridge		Dit is het bridge script waarin enkele andere scripts aangeroepen kunnen wordenmysql.sh			Maakt mysql database en/of bijbehorende user met rechten aan indien die niet bestaansettings			Diverse settings nodig voor deployment (zoals wie is de apache user, waar staat de httpd deamon ctl)symfony2acl.sh		Zet de rechten van een symfony deployment goed van logs en cachetestsettings.sh		Checkt of de settings in settings goed staan
